# toDoList

This is a project for the Mobile Device Programming class

It is a to-do list where you can add tasks, delete them, or update them. A task contains:
- title
- description
- creation time and due time
- status (completed/not completed)
- category
- notification (enabled/disabled). If enabled, it should appear before the deadline and by clicking on it, the app and specific task open for editing.
- attachments, like a photo or file (optional). Available options to remove them, add them, and to show how many it has.

The app has the options to: 
- hide completed tasks
- select what categories are visible from the list
- choose how many minutes before the deadline the notification should appear.
It also has a search bar on top and the tasks are ordered by due time, with the most urgent ones on top.
